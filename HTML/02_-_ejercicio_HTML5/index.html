¿Por Qué Recopilamos Datos al Azar?

En este proyecto, nos interesa la diversidad de las respuestas humanas cuando se les presentan preguntas inesperadas. No hay respuestas correctas o incorrectas, solo tu perspectiva única.

Algunos temas que nos interesan:

  • Colores inventos inesperados
  • Sonidos que te hacen sonreír
  • Objetos aleatorios que te recuerden algo
  • El clima ideal para una siesta

Una representación visual de la diversidad de datos

Instrucciones para Participar:

  1. Lee cada pregunta con atención.
  2. Responde con lo primero que se venga a la mente, ¡no lo pienses demasiado!
  3. Haz clic en "Enviar Datos" cuando hayas terminado.
